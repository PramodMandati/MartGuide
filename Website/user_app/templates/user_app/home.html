MartGuide

YRP Tech provides a application for shops. The application is useful for any type of shops. The features of the application is to storing the data of the items list in the shop and their prices. The shop owners can calculate the total amount of the items purchased by the customers. The application also shows the items which are taken with other items. By these the shop owners can keep offers on the items which are taken by the other items. These will help in the increase of sales.
